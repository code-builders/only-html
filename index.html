Today at Code Builders: HTML

Date: Monday, October 26, 2015

Today at Code Builders, we left the Ruby world for a quick trip to the lands of HTML and CSS. It's incredible to think of what we've learned in 3 short weeks. HTML was the first language that I began to learn on my coding journey 2 years ago. Going back to it after the 3 weeks of Ruby makes this seem like a piece of cake, which is super cool. There's still a limitless potential of stylization with CSS which we'll be learning about tomorrow and combining together the remainder of the week.
